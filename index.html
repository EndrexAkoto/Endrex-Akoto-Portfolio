SkillHive

Overview: SkillHive is a collaborative portfolio project developed by a team of three software engineers, showcasing our skills, experiences, and projects.

Features:

  • Create and customize personal portfolios for different domains of software development.
  • Showcase individual projects, skills, and experiences.
  • Collaborative project sections highlighting team contributions.
  • Interactive visualization of project statistics and achievements.

Technologies Used: Node.js, Express, MongoDB, Redis (Backend); HTML, CSS, JavaScript, Bootstrap, SCSS, LESS (Frontend)

Getting Started:

git clone https://github.com/EndrexAkoto/SkillHive.git
yarn install
yarn dev

Visit localhost:3000 to preview locally.

GitHub Stats App

Overview: A data visualization tool for retrieving and displaying GitHub repository information and commit statistics.

Features:

  • Retrieve and display repository metrics (stars, forks, watchers).
  • Analyze contributors and their contributions.
  • Visualize commit history to identify patterns.
  • Provide codebase statistics, including lines of code and programming languages used.
  • Track pull requests, issues, and branches.

Technologies Used: Python, Flask (Backend); HTML, CSS, JavaScript, Bootstrap (Frontend)

Getting Started:

git clone https://github.com/EndrexAkoto/github_stats_app.git
pip install virtualenv
cd github_stats_app
python -m venv venv
source venv/bin/activate (or venv\Scripts\activate on Windows)
pip install -r requirements.txt
flask run

Visit localhost:5000 to preview locally.

Live Demo: View Live Demo
